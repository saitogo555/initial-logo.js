Initial Logo Playground

1. HTMLDivElement (generateLogo)

ブラウザ環境でのみ動作。CSSスタイルを適用したHTMLエレメントを生成。

2. SVGElement (generateSvgElement)

ブラウザ環境でのみ動作。DOM操作可能なSVG要素を生成。

3. SVG Data URL (generateSvgDataUrl)

imgタグのsrcやCSSのbackground-imageで使用可能なData URLを生成。

4. Raw SVG String (generateRawSvg)

汎用的なSVG文字列。Node.js環境でも使用可能。ファイル保存やメール送信などに最適。
